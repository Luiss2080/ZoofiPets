# Inicio Rápido

Guía rápida para levantar el entorno de desarrollo si ya tienes todo instalado.

## 1. Levantar Servicios

Abre tu terminal en la carpeta `System` y ejecuta:

**Opción A (Comandos por separado):**
Terminal 1 (Laravel):

```bash
php artisan serve
```

Terminal 2 (Vite - Estilos/JS):

```bash
npm run dev
```

**Opción B (Script unificado si está configurado):**
Si tienes el script `start.bat` en la raíz o un comando npm configurado:

```bash
npm run dev
# (Esto dependerá de tu configuración específica de scripts en package.json)
```

## 2. Acceder

Ve a tu navegador y abre:
[http://127.0.0.1:8000](http://127.0.0.1:8000)

## 3. Login

Usa las credenciales de administrador (ver `CREDENCIALES.md`) para acceder al panel.
